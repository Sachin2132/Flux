# Flux | Premium Web Experience

A high-end, responsive web application featuring a modern Dark Theme, glassmorphic UI, and seamless user authentication. This project demonstrates advanced front-end design techniques and secure integration with Firebase.

## ✨ Features

- **Premium Dark UI**: A sophisticated dark theme built with a curated color palette and refined typography.
- **Glassmorphism**: Elegant use of backdrop-blur effects on navigation and cards for a depth-rich experience.
- **Interactive Animations**:
  - **Hero Glow**: Rotating gradient borders on the authentication form.
  - **Success Celebration**: Integrated canvas-confetti and SVG checkmark animations on successful login.
  - **Smooth Transitions**: CSS-driven entrance animations and hover effects.
- **Firebase Authentication**: Secure user sign-up and login functionality.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Keyframes), JavaScript (ES6 Modules)
- **Backend/Auth**: Firebase Authentication
- **Icons**: [Ionicons](https://ionicons.com/)
- **Animations**: [Canvas Confetti](https://github.com/catdad/canvas-confetti)
- **Fonts**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Register a new account on the Sign Up page or log in using existing credentials.
